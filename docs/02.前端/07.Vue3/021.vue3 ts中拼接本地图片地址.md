当我们在`vue3`中使用本地的图片地址,该地址又是以`:src="地址"`这种方式使用时，不能直接该图片的地址，而是要用`vite`提供的图片引用方式
new Url('图片地址',import.meta.url).href
```vue
<script steup>
import { computed } from 'vue'
const imgPath = computed(() => {
  return new URL('', import.meta.url).href
})
</script>
<template>
  <img :src="imgPath()"> 
</template>
```
